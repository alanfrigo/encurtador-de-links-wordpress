Encurta URLs usando a API Kutt.it e permite gerenciar links diretamente do painel do WordPress.

== Description ==

O plugin Encurtador de Links integra-se com a API Kutt.it para fornecer uma solução de encurtamento de URLs diretamente no painel de administração do WordPress. Com ele, você pode criar URLs curtas, gerenciar suas URLs encurtadas e configurar parâmetros UTM para rastreamento.

= Funcionalidades principais =
* Encurtamento de URLs usando a API Kutt.it
* Gerenciamento de URLs encurtadas
* Configuração de parâmetros UTM
* Verificação da conexão com a API Kutt.it

== Installation ==

1. Faça o upload dos arquivos do plugin para o diretório `/wp-content/plugins/encurtador-de-links` ou instale o plugin diretamente através da tela de plugins do WordPress.
2. Ative o plugin através da tela 'Plugins' no WordPress.
3. Acesse o menu 'Encurtador de Links' no painel de administração do WordPress para configurar o plugin.
4. Insira a URL da API e a chave da API fornecidas pelo Kutt.it.

== Frequently Asked Questions ==

= Como obtenho a chave da API do Kutt.it? =
Você pode obter a chave da API registrando-se no Kutt.it e gerando uma chave de API na seção de configurações de sua conta.

= O plugin é gratuito? =
Sim, o plugin é gratuito e de código aberto. No entanto, o serviço Kutt.it pode ter suas próprias limitações e políticas de uso.

= Quais são os requisitos mínimos para usar o plugin? =
O plugin requer WordPress 5.0 ou superior e PHP 7.2 ou superior.
